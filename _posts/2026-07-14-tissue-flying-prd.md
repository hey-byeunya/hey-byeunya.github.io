## 무엇을

화면 아래쪽을 터치해서 바람을 일으켜, 떨어지는 티슈를 최대한 오래 공중에 띄우는 게임.

## 누가

모바일 웹 브라우저를 쓰는 누구나. 설치 없이 링크만 열면 바로 플레이할 수 있다.

## 규칙과 끝나는 조건

### 기본 규칙

- 티슈가 화면 중앙 위쪽에서 떨어지기 시작한다.
- 화면 아무 곳이나 터치하면 그 위치에서 티슈 쪽으로 바람이 분다.
- 손가락을 떼지 않고 드래그해도 계속 바람이 나간다.
- 티슈가 공중에 떠 있는 시간(초)이 곧 점수다.
- 역대 최고 기록은 자동 저장된다.

### 시간이 지날수록 어려워진다

| 시간 | 변화 |
|------|------|
| 0–10초 | 없음. 조작에 익숙해지는 구간 |
| 10초~ | 랜덤 돌풍이 불어 티슈를 밀어냄 |
| 15초~ | 바닥이 서서히 올라옴 (놀 수 있는 공간이 줄어듦) |
| 20초~ | 빨간 위험 구간 등장, 들어가면 티슈가 빠르게 끌려 내려감 |
| 30초~ | 중력이 점점 세져서 바람을 더 세게 불어야 함 |

### 끝나는 조건

**티슈가 바닥에 닿으면 게임 오버.** 그 외에 시간 제한이나 목숨 같은 건 없다.

## 화면 요소

### 시작 화면 (게임 시작 전)

- 티슈 그림과 "여기를 터치하세요" 안내 애니메이션
- 화면 아무 곳이나 터치하면 카운트다운(3-2-1) 후 게임 시작

### 플레이 중

- **티슈** — 하늘하늘 흔들리며 떨어지는 흰 티슈
- **타이머** — 좌측 상단에 체공 시간 표시 (예: 12.5초)
- **최고 기록** — 우측 상단에 Best 시간 표시
- **단계 안내** — 돌풍이 시작되면 "돌풍 주의", 위험해지면 "위험 구간" 등 텍스트 표시
- **위험 구간** — 빨간 띠로 표시되고, "DANGER" 라벨이 붙음
- **바닥 상승** — 올라오는 바닥은 빨간 줄무늬로 표시
- **바닥 접근 경고** — 티슈가 바닥 가까이 가면 화면 아래가 빨갛게 빛남

### 게임 오버 화면

- 이번 판 기록 (큰 글씨)
- 신기록이면 "New Best!" 금색 텍스트
- 역대 최고 기록 (짙은 회색 박스 안에 흰 글씨로 강조)
- 화면 터치하면 바로 재시작

## 예외 상황

| 상황 | 동작 |
|------|------|
| 게임 오버 상태에서 터치하면? | 바로 카운트다운이 시작되고 새 게임이 시작된다 |
| 카운트다운 중에 터치하면? | 아무 일도 일어나지 않는다. 카운트다운이 끝나야 조작 가능 |
| 티슈를 벽 구석에 몰아넣으면? | 벽·천장 근처에서 불규칙한 바람이 불어 구석에 머물 수 없다 |
| 티슈가 화면 밖으로 나가면? | 벽과 천장에 부딪혀 튕겨 나온다. 화면 밖으로 사라지지 않는다 |
| 바닥이 계속 올라오면 공간이 없어지나? | 최소 높이(150px)가 보장되어 완전히 닫히지는 않는다 |
| 브라우저를 닫았다 다시 열면? | 최고 기록은 유지된다. 진행 중이던 게임은 처음부터 다시 시작 |
| 여러 손가락으로 동시에 터치하면? | 마지막으로 터치한 위치에서만 바람이 분다 |
